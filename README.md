# KSP-Tel
KSP Telemetry 

# Requirements
    * Mat Plot Lib
    * Python
    * KOS
# Step by Step Tutorial
    -1. Install Python 3 at https://www.python.org/
    -2. Download the KOS mod for KSP (Ckan or manually)
    -3. Download the Code from the github repository
    -4. Extract the ZIP file and move the file named 'KSP Tel' to the desktop
    -5. Launch ksp once (Only needed if you have not installed kos yet)
    -6. Copy the file 'logTel.ks' to the script file (Kerbal Space Program/Ships/Script)
    -7. Launch ksp and take your craft to the launch pad.
    -8. Scroll on the toolbar until you see the kos icon, click on it
    -9. A terminal should appear, in it type 'switch to 0.' then press enter
    -10. After you have completed step 9 type 'run logTel.' and press enter
    -11. Press the 1 key on your keyboard to activate action group 1
    -12. Launch your rocket normally, and once you are in orbit press 0 to activate action group 10
    -13. The script folder should now be full of files, move them all into the telemetry folder, which is located inside of the 'KSP Tel' folder
    -14. Open up command prompt
    -15. In file explorer go to the top bar where it says which folder you are viewing, click on the bar, once everything is highlighted in blue click 'Control + C' to copy
    -16. Go back into command prompt and type 'CD' then press space and press 'Control + V' to paste in the file path you copied press enter
    -17. Now type 'python3 main.py' and press enter
    -18. 2 things should now show up on the comnand prompt window 'start' and 'exit'
    -19. Go to the line just below exit and type '1'
    -20. Then you will see a list of numbers and corosponding files, type the number and press enter on the one you would like to view
    -21. If you have done everything correctly, a window should open showing you a graph, that is you telemetry
    -22. Enjoy!
